Análise de Atrito nas Equipes de Marketing
Visão Geral
Este projeto tem como objetivo analisar os dados relacionados ao atrito nas equipes de marketing da empresa, em colaboração com o Parceiro de Negócios de Recursos Humanos (HRBP). O objetivo é identificar as causas do atrito e apresentar descobertas e insights para melhorar a retenção de talentos.

Dados
Os dados utilizados estão contidos no arquivo dataset.xlsx, disponibilizado internamente. O dataset inclui informações sobre funcionários das equipes de marketing, motivos de saída da empresa, avaliações de desempenho e outras métricas relevantes.

Metodologia
Análise Exploratória de Dados: Explorar os dados para entender a distribuição, tendências e padrões.
Limpeza e Transformação de Dados: Tratar dados ausentes, inconsistências e criar novas variáveis, se necessário.
Análise de Padrões: Identificar padrões nos dados que possam estar relacionados ao atrito nas equipes de marketing.
Visualizações: Criar gráficos e visualizações para apresentar os resultados de forma clara e impactante.
Insights e Recomendações: Comentar sobre os principais insights e fornecer recomendações para melhorar a retenção de talentos.
Resultados
Os resultados da análise incluem insights sobre os principais motivos de atrito nas equipes de marketing, a relação entre o desempenho e a saída dos funcionários, e recomendações para ações futuras.

Conclusão
Este projeto fornece uma análise detalhada do atrito nas equipes de marketing e sugere medidas para melhorar a retenção de talentos. Espera-se que essas informações sejam úteis para a equipe de RH e gestores de marketing na tomada de decisões estratégicas.
